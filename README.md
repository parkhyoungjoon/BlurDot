# BlurDot

K-Digital 과정 기업프로젝트 4조   
영상 비식별화 AI 서비스 BlurDot의 서비스를 지원하는 웹 사이트 개발

## 시작하기에 앞서

본 프로젝트는 개발공부 및 테스트 목적으로 로컬환경에서 실행되는 프로젝트 입니다.

### 개발환경

- JDK : 21
- Server : tomcat-embed-core-10.1.31 (Spring Boot 내장 서버)
- DB : MariaDB
- jdbc : mariadb-java-client-3.3.3

### 사용 라이브러리
Spring Boot, Jquery, Bootstrap

## 테스트

로컬환경에서 테스트 진행
2024-11-22 - 문제없이 프론트엔드 화면 출력 

## 전개

BlurDot/blurdot/src/main/java/com/mysite/blurdot : Controller, Model 위치
blurdot/src/main/resources/templates : 웹 사용될 템플릿 위치 View 
blurdot/src/main/resources/static : 웹에서 사용할 이미지, css , js 위치
